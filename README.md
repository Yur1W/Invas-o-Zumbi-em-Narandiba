## Overview
This project aims to demonstrate the utilities of the 2DTileset Mapper function in Unity 2022, this sistem can be essential for the creation and alteration of game levels and environments, proving to be an essencial asset in a Game Developer`s toolkit.

## What is the 2DTileset Mapper function?
A paralax system is a visual technique used in 2D games which utilizes Scripts and clever asset positioning to create a sense of depth and motion in a game.

## Implementation in Project
In this project the Unity Paralax System was implemented in 1 main way:

### 
- Creating the feel of a never ending scenery.
  
### 
- Giving depth to the scenery and making it feel less stiff.
